# ytp-rotate - 油管视频旋转脚本

油管视频旋转插件。Tampermonkey脚本。 

## 功能

- 旋转视频播放器视角
- 水平翻转视频
- 垂直翻转视频
- 打开和关闭全屏填充模式
- 打开和关闭画中画模式

## 用法

### 快捷按钮

在播放器右侧控制栏添加了两个快捷按钮:

- 旋转视频按钮: 点击可顺时针旋转视频0度
- 填充屏幕按钮: 点击可切换全屏填充模式

![buttons](https://github.com/zhzLuke96/ytp-rotate/raw/master/docs/btns.png)

> 红框内为本脚本添加内容

### 右键菜单

在视频播放器上右键,可以看到添加的选项:

- 开/关插件: 可以关闭插件避免和其它脚本冲突
- 旋转90°: 显示/切换 旋转角度
- 填充屏幕: 显示/切换 全屏填充模式
- 水平翻转: 显示/切换 水平翻转
- 垂直翻转: 显示/切换 垂直翻转  
- 画中画: 显示/切换 在画中画模式
